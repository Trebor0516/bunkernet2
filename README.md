# **BUNKERNET**

Sistema de chat colaborativo en tiempo real utilizando WebSocket.

## **Descripcion**

Aplicacion web que permite la comunicacion real entre multiples usuarios mediante WebSocket.

El sistema permite:
  - Envio y recepcion de mensajes en tiempo real.
  - Historial visible de mensajes.
  - Conexion simultanea de multiples usuarios.
  - Notificacion de usuarios conectados y desconectados
  - Asignacion automatica de nombres temporales.
